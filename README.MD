# 🎓 School Management System

Un système de gestion scolaire développé ,  permettant de gérer les étudiants, les professeurs, les matières, les notes, les absences et les statistiques.

## Fonctionnalités

### Authentification
- Connexion des utilisateurs
- Gestion des rôles (Administrateur, Professeur, Étudiant)

### Gestion des étudiants
- Ajouter un étudiant
- Afficher les étudiants
- Modifier un étudiant
- Supprimer un étudiant

### Gestion des professeurs
- Ajouter un professeur
- Afficher les professeurs
- Modifier un professeur
- Supprimer un professeur

### Gestion des matières
- Ajouter une matière
- Afficher les matières
- Modifier une matière
- Supprimer une matière

### Gestion des notes
- Ajouter une note
- Afficher les notes
- Modifier une note
- Supprimer une note

### Gestion des absences
- Enregistrer une absence
- Consulter les absences

### Statistiques
- Moyenne générale
- Moyenne par étudiant
- Meilleur étudiant
- Nombre d'absences

## Installation

1. Cloner le projet

```bash
git clone https://github.com/votre-utilisateur/school-management.git
```

2. Accéder au dossier

```bash
cd school-management
```

3. Lancer le projet

```bash
python3 main.py
```
